# 宏村水脉 Project Bible v2.1

# DATA_SCHEMA_07_Social.md

版本： Project Bible v2.1

文档类型： Data Schema Design Document

用途：

定义《宏村水脉》中的社会治理、人口、经济、水规和维护系统数据结构。

------------------------------------------------------------------------

# 1. 系统定位

Social系统用于模拟：

-   宗族治理；
-   人口变化；
-   资源积累；
-   公共工程组织；
-   水规执行；
-   长期维护。

核心思想：

宏村水利不是一次性工程，而是一套持续运行的社会系统。

------------------------------------------------------------------------

# 2. 社会运行模型

    人口

    ↓

    需求

    ↓

    资源

    ↓

    公共工程

    ↓

    规则

    ↓

    维护

    ↓

    长期稳定

------------------------------------------------------------------------

# 3. 人口系统 Population

人口影响：

-   住房需求；
-   用水压力；
-   劳动力；
-   工程能力。

示例：

``` json
{
"id":"population",
"value":300,
"labor":120,
"waterDemand":80
}
```

------------------------------------------------------------------------

# 4. 经济系统 Economy

资源来源：

-   农业；
-   商贸；
-   手工业；
-   家族财富。

示例：

``` json
{
"wealth":500,
"income":50,
"publicFund":100
}
```

------------------------------------------------------------------------

# 5. 公共资金系统

大型工程需要：

    资源

    +

    劳动力

    +

    社会支持

例如：

南湖建设需要：

-   财富积累；
-   人口基础；
-   工程需求；
-   组织能力。

------------------------------------------------------------------------

# 6. 治理系统 Governance

核心组织：

汪氏宗族。

功能：

-   工程决策；
-   资源分配；
-   冲突协调；
-   公共事务管理。

示例：

``` json
{
"organization":"wang_clan",
"authority":80,
"trust":90
}
```

------------------------------------------------------------------------

# 7. 水规系统 Water Rules

模拟传统村落用水秩序。

包括：

-   饮用；
-   洗涤；
-   灌溉；
-   消防储备。

示例：

``` json
{
"rule":"daily_water_order",
"priority":[
"drinking",
"cleaning",
"irrigation"
]
}
```

------------------------------------------------------------------------

# 8. 劳动力系统 Labor

公共工程需要：

-   工匠；
-   村民；
-   管理者。

示例：

``` json
{
"workers":200,
"available":80,
"skillLevel":60
}
```

------------------------------------------------------------------------

# 9. 维护系统 Maintenance

水利建成后需要：
