# 宏村水脉

# Hongcun Water Pulse

# GDD06_Social_Economy_System.md

版本： Project Bible v2.1

文档类型： Game Design Document

用途：

定义《宏村水脉》中的社会治理、经济循环、人口增长、公共工程和村庄运行系统。

核心思想：

宏村水利体系能够持续运行，不仅依靠工程技术，更依靠长期形成的社会组织和公共规则。

------------------------------------------------------------------------

# 1. 系统目标

社会系统负责模拟：

-   村庄如何组织；
-   工程如何筹集资源；
-   居民如何共同维护水系统；
-   公共设施如何随着需求发展。

核心逻辑：

    人口

    ↓

    需求

    ↓

    资源

    ↓

    工程

    ↓

    规则

    ↓

    长期维护

------------------------------------------------------------------------

# 2. 社会系统设计原则

## 原则一：公共工程依靠社会协作

玩家不能直接创造大型工程。

建设需要：

-   资源；
-   劳动力；
-   社会支持；
-   工程条件。

------------------------------------------------------------------------

## 原则二：维护比建设更重要

工程完成不是结束。

长期运行需要：

-   清淤；
-   修复；
-   管理；
-   水规执行。

------------------------------------------------------------------------

# 3. 宗族治理系统

核心组织：

## 汪氏宗族

功能：

-   发起公共工程；
-   组织资源；
-   协调居民关系；
-   维护公共秩序。

宗族不是简单权力系统，而是传统村落公共事务组织形式。

------------------------------------------------------------------------

# 4. 人口系统

人口变化：

    出生

    ↓

    家庭增加

    ↓

    住房需求

    ↓

    水资源压力

    ↓

    村庄扩展

人口影响：

-   用水量；
-   劳动力；
-   工程需求；
-   消防压力。

------------------------------------------------------------------------

# 5. 经济系统

本系统模拟传统乡村资源循环。

## 资源来源：

包括：

-   农业生产；
-   手工业；
-   商贸积累。

------------------------------------------------------------------------

## 公共资源：

用于：

-   修筑堨坝；
-   维护水圳；
-   建造桥梁；
-   扩建南湖。

------------------------------------------------------------------------

# 6. 公共工程系统

大型工程必须满足：

    资源

    +

    劳动力

    +

    社会支持

    +

    工程条件

例如：

南湖建设需要：

-   足够人口；
-   经济积累；
-   水系统需求；
-   组织能力。

------------------------------------------------------------------------

# 7. 水规系统

水利运行依靠规则。

玩家需要管理：

-   饮用水；
-   洗涤用水；
-   灌溉用水；
-   消防储备。

目标：

协调：

个人需求

与

公共利益。

------------------------------------------------------------------------

# 8. 村民满意度系统

影响：

-   工程支持度；
-   维护积极性；
-   社会稳定。

示例：

水圳设计合理：

↓

居民满意提高

↓

维护效率提高。

------------------------------------------------------------------------

# 9. 劳动力系统

公共工程需要：

-   工匠；
-   村民；
-   管理人员。

玩家需要安排：

-   修建时间；
-   清淤周期；
-   住宅建设；
-   公共工程顺序。

------------------------------------------------------------------------

# 10. 灾害社会响应

## 洪水

影响因素：

-   工程质量；
-   管理能力；
-   储水设施。

------------------------------------------------------------------------

## 火灾

影响因素：

-   水源布局；
-   消防路径；
-   社区协作。

------------------------------------------------------------------------

# 11. 南湖解锁逻辑

南湖不是时间自动出现。

需要：

    人口增长

    +

    经济积累

    +

    水系统压力

    +

    社会组织能力

满足条件后：

开启公共工程任务。

# 15. 验收标准

完成后：

玩家可以理解：

-   为什么村民需要合作；
-   为什么工程需要资金；
-   为什么维护比建设更重要；
-   为什么南湖需要长期积累。

系统：

-   数据驱动；
-   与水系统连接；
-   支持历史阶段变化。

------------------------------------------------------------------------

# 16. 设计总结

宏村水利的核心不是一条水渠。

而是一套：

工程技术、

社会组织、

公共规则、

长期维护

共同形成的生命系统。
