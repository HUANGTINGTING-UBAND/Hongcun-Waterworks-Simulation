英文： Hongcun Water Pulse

类型： Historical Simulation Game

历史水利模拟互动游戏。

------------------------------------------------------------------------

# 2. 核心概念

《宏村水脉》模拟安徽宏村从自然环境形成，到水利工程建设，再到聚落成熟发展的全过程。

玩家不是创造一个不存在的城市，而是在真实地理、水文、工程和社会规则约束下参与宏村形成过程。

核心逻辑：

自然环境

↓

水资源

↓

水利工程

↓

居民生活

↓

建筑形成

↓

社会制度

↓

文化遗产

------------------------------------------------------------------------

# 3. 玩家身份

玩家身份：

历史水利规划参与者。

不是：

-   上帝视角城市创造者；
-   虚构文明建立者。

玩家参与：

-   地形观察；
-   水利规划；
-   工程建设；
-   村庄维护。

------------------------------------------------------------------------

# 4. 玩家成长路径

## 阶段一：观察者

目标：

理解土地。

能力：

-   查看地形；
-   分析水流；
-   学习环境。

------------------------------------------------------------------------

## 阶段二：规划助手

目标：

协助水利设计。

能力：

-   地形测绘；
-   水路规划；
-   工程选择。

------------------------------------------------------------------------

## 阶段三：工程参与者

目标：

推动建设。

能力：

-   调配资源；
-   管理劳动力；
-   解决工程问题。

------------------------------------------------------------------------

## 阶段四：村庄维护者

目标：

保证系统长期运行。

能力：

-   水规管理；
-   灾害应对；
-   聚落调整。

------------------------------------------------------------------------

# 8. 历史真实性原则

所有内容区分：

A： 明确历史事实

B： 多来源记载

C： 地方文化解释

D： 游戏设计推演

禁止：

-   将游戏推演伪装成历史事实；
-   修改宏村真实发展逻辑；
-   随意创造历史人物。

-   进入宏村世界；
-   查看基础地形；

# 13. 设计总结

《宏村水脉》的核心不是建造一个漂亮村庄。

而是让玩家理解：

一个村庄如何通过水、工程和社会协作，在时间中形成生命。
