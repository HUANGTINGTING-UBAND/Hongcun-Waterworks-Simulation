# 宏村水脉 Project Bible v2.1

# VISUAL BIBLE 01：总体视觉定位

版本：v2.1

文档类型：Visual Design Specification

## 1. 项目视觉定位

《宏村水脉》定位为：

> 一个可进入的数字历史遗产沙盘。

关键词：

-   东方数字遗产
-   历史模拟
-   水墨写实
-   微缩景观
-   动态时间演化

## 2. 核心视觉目标

玩家需要感受到：

-   水如何塑造村庄；
-   建筑如何围绕水成长；
-   社会如何维护水系统。

## 3. 风格方向

### 数字沙盘感

特点： - 上帝视角 - 可旋转 - 可缩放 - 强空间关系

### 中国传统美学

色彩： - 灰白墙 - 深灰瓦 - 青绿色水 - 土黄色山体 - 淡雾天空

光照： - 柔和 - 晨雾 - 阴天散射光

避免： - 欧美HDR风格 - 过度游戏化色彩

### 建筑表现

目标： 高质量数字模型。

重点： - 马头墙轮廓 - 黑瓦层次 - 白墙材质 - 水面倒影

## 4. 摄像机设计

默认： 45度俯视角。

支持： - 鸟瞰模式 - 工程模式 - 漫游模式

## 5. 水体表现

水是项目核心叙事元素。

自然溪流： - 自然流动 - 较快流速

水圳： - 狭窄 - 清澈 - 缓慢

月沼： - 平静 - 建筑倒影

南湖： - 开阔 - 调蓄能力

## 6. 资产等级

LOD3： 远景村庄。

LOD2： 中景建筑。

LOD1： 近景交互。

## 8. 第一阶段视觉验收

展示： - 雷岗山 - 西溪 - 月沼 - 南湖 - 建筑群 - 四季变化 - 历史阶段切换

任何视觉开发必须优先服务：

"让用户理解一条水如何改变一个村庄。"
