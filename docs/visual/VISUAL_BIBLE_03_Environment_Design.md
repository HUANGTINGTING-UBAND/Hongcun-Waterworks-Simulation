# 宏村水脉 Project Bible v2.1

# VISUAL BIBLE 03：Environment Design 环境设计规范

版本： Project Bible v2.1

文档类型： Visual Environment Guideline

用途：

定义《宏村水脉》游戏世界中的地形、水体、聚落环境、天气、季节和镜头表现规范。

------------------------------------------------------------------------

# 1. 环境设计核心目标

玩家进入游戏第一分钟，需要理解：

"为什么这里会成为宏村？"

而不是：

"这里有很多徽派建筑。"

环境优先级：

    地形

    ↓

    水

    ↓

    聚落

    ↓

    建筑

    ↓

    装饰

------------------------------------------------------------------------

# 2. 雷岗山环境设计

定位：

宏村空间背景骨架。

作用：

-   提供靠山关系；
-   体现徽州山水格局；
-   表现水源方向。

视觉：

颜色：

-   深青绿；
-   灰绿色山体；
-   低饱和雾气。

形态：

应表现：

-   徽州丘陵；
-   缓坡；
-   连续山体。

避免：

-   高耸雪山；
-   戏剧化山峰。

------------------------------------------------------------------------

# 3. 西溪环境设计

定位：

宏村生命起点。

核心：

表现自然溪流如何被人工水利系统重新组织。

------------------------------------------------------------------------

## Stage 0

自然状态：

-   河道自由；
-   岸线自然；
-   水流不规则。

------------------------------------------------------------------------

## 后期阶段

人工控制：

-   堨坝；
-   引水入口；
-   分流结构。

------------------------------------------------------------------------

# 4. 水圳环境设计

水圳是游戏核心视觉元素。

特点：

    细

    长

    缓慢

    穿过村庄

表现：

-   石砌渠壁；
-   青苔；
-   水面反光；
-   与建筑融合。

------------------------------------------------------------------------

# 5. 月沼环境设计

定位：

宏村核心公共空间。

视觉关键词：

    静

    圆

    聚

    公共

表现：

-   半月形水面；
-   白墙倒影；
-   村民活动。

时间变化：

清晨：

-   雾气；
-   柔光。

白天：

-   建筑倒影。

夜晚：

-   灯火映水。

------------------------------------------------------------------------

# 6. 南湖环境设计

定位：

后期水利扩展。

区别：

月沼：

    村中心

    生活空间

南湖：

    村外

    调节空间

    景观节点

表现：

-   开阔水面；
-   山水关系；
-   蓄水功能。

------------------------------------------------------------------------

# 7. 四季系统

## 春

关键词：

-   新绿；
-   湿润；
-   生长。

## 夏

关键词：

-   雨；
-   高水量；
-   荷叶。

## 秋

关键词：

-   金黄；
-   收获；
-   村落宁静。

## 冬

关键词：

-   灰白；
-   薄雾；
-   静谧。

------------------------------------------------------------------------

# 8. 天气系统

第一阶段：

实现：

-   晴天；
-   雾；
-   小雨。

第二阶段：

增加：

-   暴雨；
-   洪水事件。

------------------------------------------------------------------------

# 9. 宏村八景系统

不要作为简单旅游列表。

设计为：

空间探索和历史解锁奖励。

例如：

完成水系建设后：

解锁：

-   月沼夜景；
-   南湖春晓。

------------------------------------------------------------------------

# 10. 镜头设计

默认：

45度鸟瞰视角。

用于：

理解：

-   山；
-   水；
-   聚落关系。

任务模式：

降低镜头高度。

观察：

-   水圳；
-   建筑；
-   工程细节。

------------------------------------------------------------------------

# 11. 第一阶段环境资产优先级

## P0

必须：

-   雷岗山；
-   西溪；
-   基础地形；
-   水面；
-   雾。

## P1

增加：

-   月沼；
-   基础徽派建筑；
-   桥。

## P2

增加：

-   南湖；
-   完整街巷；
-   民居细节。

------------------------------------------------------------------------

# 12. 环境验收标准

Vertical Slice必须达到：

玩家打开网页后：

3秒内理解：

-   这是徽州；
-   这里有山；
-   这里有水；
-   这里未来会形成宏村。

------------------------------------------------------------------------

# 13. 总结

环境设计服务于：

山水关系、

水利演化、

聚落形成、

历史体验。

《宏村水脉》的世界不是建筑集合，而是一套由自然、水和人共同形成的生命系统。

END
