# 项目身份

你现在接手：

《宏村水脉》（Hongcun Water Pulse）

这是一个基于真实安徽宏村历史水利体系的网页互动历史模拟项目。

目标：

通过数字化方式展示：

-   地形如何影响聚落；
-   水利如何塑造村庄；
-   社会如何维护工程；
-   人与自然如何共同形成文化遗产。

# 技术约束

项目技术路线：

-   React
-   TypeScript
-   Vite
-   Three.js
-   React Three Fiber

禁止：

-   改为 Unity/C# 架构；
-   引入与项目方向无关的大型框架；
-   删除已有设计文档；
-   绕过数据结构直接硬编码游戏逻辑。

------------------------------------------------------------------------

# 历史真实性规则

必须遵守：

1.  不虚构历史人物。

2.  不修改核心历史时间线。

3.  不将民间传说作为确定史实。

4.  所有新增历史内容必须标记可信等级。

------------------------------------------------------------------------

# 开发原则

采用：

Data Driven Architecture

规则：

    数据
     ↓
    系统
     ↓
    表现

不要：

    代码
     ↓
    隐藏数据
