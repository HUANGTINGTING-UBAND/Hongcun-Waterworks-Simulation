# 1. 项目简介

《宏村水脉》（Hongcun Water
Pulse）是一款基于安徽宏村真实历史环境、水利体系和聚落演化过程设计的历史模拟互动游戏。

玩家不是创造一个虚构村庄，而是在真实地理、水文、工程和社会规则约束下，参与宏村水利体系形成与发展的全过程。

玩家将经历：

    自然环境观察
    ↓
    村址选择
    ↓
    地形勘测
    ↓
    水利规划
    ↓
    堨坝建设
    ↓
    水圳形成
    ↓
    月沼建设
    ↓
    民居演化
    ↓
    水规建立
    ↓
    村庄扩展
    ↓
    南湖形成
    ↓
    遗产维护

核心理念：

> 宏村并不是因为建筑而存在，而是一套持续运行的水系统塑造了建筑、生活和文化。

# 2. 核心设计理念

## 2.1 历史真实性优先

所有内容采用：

  等级   定义                 用途
  ------ -------------------- ----------------
  A      明确历史事实         核心剧情与系统
  B      多来源记载           辅助设计
  C      地方记忆或文化解释   氛围叙事
  D      游戏模拟推演         玩法机制

禁止： - 无依据创造历史事件； - 将传说人物作为历史事实； -
改变宏村真实发展逻辑。

------------------------------------------------------------------------

# 3. 游戏核心系统

## 3.1 水利系统

宏村水利体系：

    自然溪流
    ↓
    引水工程
    ↓
    村内水圳
    ↓
    月沼调蓄
    ↓
    南湖扩展
    ↓
    农业与生活循环

玩家学习：

-   为什么这里适合建村；
-   如何利用自然高差；
-   如何设计稳定水循环。

------------------------------------------------------------------------

## 3.2 地形系统 Terrain System

负责：

-   雷岗山；
-   河谷；
-   高程；
-   坡度；
-   可建设区域。

数据：

    data/terrain/

------------------------------------------------------------------------

## 3.3 聚落系统 Settlement System

负责：

-   民居；
-   街巷；
-   天井；
-   庭院水体。

建筑不是独立资产，而必须回应：

-   水源；
-   地形；
-   消防；
-   居民需求。

------------------------------------------------------------------------

## 3.4 社会系统 Society System

模拟：

-   人口增长；
-   劳动力；
-   公共工程；
-   水规；
-   清淤维护。

核心：

> 工程完成不是结束，维护才是系统延续的开始。

------------------------------------------------------------------------

# 10. 项目愿景

《宏村水脉》的目标不是复制一个旅游景区。

而是创造一个：

> 可以让用户亲身理解中国传统聚落智慧的数字历史世界。

让用户看到：

一条水，如何改变土地；

一个村庄，如何依靠制度延续；

人与自然，如何共同创造文化遗产。
